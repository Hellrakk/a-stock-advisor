# A股量化系统 Skill

这是一个完整的A股量化交易系统助手，通过统一命令行入口 `a_stock_manager.py` 提供全流程量化投资功能。

## 系统架构

系统遵循量化投资五阶段流程：
```
数据工程 → 因子研发 → 策略开发 → 回测验证 → 实盘工程
```

## 核心功能模块

### 1. 数据工程（6个子功能）
- **数据更新** ⭐推荐 - AKShare/BaoStock真实数据获取，增量更新
- **数据质量检查** - Pipeline检查+清洗+验证，自动修复
- **数据时效性检查** - 数据新鲜度监控
- **多源数据获取** - 智兔数服/腾讯/新浪/AKShare多源切换
- **另类数据框架** - 北向资金、融资融券、分析师研报
- **数据准备状态** ⭐新增 - 一键检查数据就绪状态

### 2. 因子研发（7个子功能）
- **因子挖掘系统** ⭐推荐 - 人工挖掘+遗传规划自动挖掘+评估验证
- **因子回测验证** - 单因子/多因子回测，股票级别分析
- **因子库管理** - 因子启用/禁用，表现分析
- **因子监控** - IC/IR下降监控，相关性监控
- **创新实验室** - 因子原型验证，创新周报
- **技术指标验证** ⭐新增 - RSI/MACD/布林带有效性验证
- **RDAgent因子挖掘** ⭐AI驱动 - 微软RDAgent自动发现

### 3. 策略开发（6个子功能）
- **多因子模型** ⭐推荐 - 多因子选股，动态权重
- **Alpha选股器** - Alpha因子股票筛选
- **市场状态识别** - 牛市/熊市/震荡市识别
- **再平衡策略** - 定期/阈值/信号驱动再平衡
- **强化学习优化器** - DQN/PPO/A2C策略优化
- **ML因子组合器** - 随机森林/XGBoost/神经网络

### 4. 回测验证（6个子功能）
- **运行回测** ⭐推荐 - 完整回测引擎，绩效分析
- **Brinson归因分析** - 配置/选择/交互效应分解
- **滚动性能分析** - 滚动收益率/波动率/夏普比率
- **压力测试** - 2008/2015/2020极端情景测试
- **过拟合检测** - 样本外验证，参数敏感性
- **绩效对比** - 因子/策略/指标对比

### 5. 实盘工程（9个子功能）
- **每日主控流程** ⭐推荐 - 完整流水线一键执行
- **推送系统** - 盘前推送(8:00)、日报推送(18:30)
- **持仓管理** - 凯利公式仓位，止损止盈，手动修改 ⭐新增
- **风控系统** - 策略容量，冲击成本，流动性风险
- **资金管理** - 动态资金分配，风险预算
- **风险预警** - 净值/回撤/仓位/因子暴露预警
- **交易员助手** - 每日/每周交易报表
- **模拟交易** - 模拟盘交易系统
- **券商API接入** - 华泰/中信/国泰君安API

### 6. 系统管理（9个子功能）
- **交易日检查** - 检查今天是否是交易日
- **系统验证** - 完整/快速验证，版本查看，Gitee更新
- **定时任务管理** - 安装/卸载/查看定时任务
- **系统配置管理** - 飞书推送配置，功能开关
- **查看日志** - 系统运行日志，健康检查报告
- **系统健康检查** - 完整健康检查
- **质量控制** - 数据质量，因子有效性验证
- **监控仪表板** - 实时监控仪表板
- **事件驱动引擎** - 异步事件处理

### 快捷入口（4个）
- **从0-1量化投资** ⭐新增 - 一键启动完整流程检查
- **每日主控流程** ⭐推荐 - 完整流水线
- **盘前推送** ⭐推荐 - 工作日8:00
- **日报推送** ⭐推荐 - 工作日18:30

## 使用方式

### 启动主入口
```bash
python3 a_stock_manager.py
```

### 推荐工作流程

#### 新用户从0-1启动
1. 选择 `7. 从0-1量化投资` - 检查系统状态
2. 选择 `1. 数据工程 -> 1. 数据更新` - 获取数据
3. 选择 `2. 因子研发 -> 1. 因子挖掘系统` - 生成因子
4. 选择 `8. 每日主控流程` - 运行完整流水线

#### 日常运维
1. 选择 `6. 系统管理 -> 6. 系统健康检查` - 检查系统状态
2. 选择 `1. 数据工程 -> 1. 数据更新` - 更新数据
3. 选择 `8. 每日主控流程` - 执行每日任务

#### 因子研发
1. 选择 `2. 因子研发 -> 1. 因子挖掘系统` - 挖掘新因子
2. 选择 `2. 因子研发 -> 2. 因子回测验证` - 验证因子
3. 选择 `2. 因子研发 -> 3. 因子库管理` - 管理因子

#### 策略优化
1. 选择 `3. 策略开发 -> 1. 多因子模型` - 调整模型
2. 选择 `4. 回测验证 -> 1. 运行回测` - 验证策略
3. 选择 `4. 回测验证 -> 5. 过拟合检测` - 检测过拟合

## 风控参数

- 个股止损：-10%
- 个股止盈：+20%
- 组合最大回撤：-15%
- 单股最大仓位：12%（实盘5%）
- 单行业最大仓位：30%

## 因子有效性标准

- IC绝对值 ≥ 0.02（弱有效）
- IC绝对值 ≥ 0.05（有效）
- IR ≥ 0.5（可接受）
- IR ≥ 1.0（良好）
- 样本量 ≥ 100只股票

## 常见问题

### 数据过期
运行 `1. 数据工程 -> 1. 数据更新` 获取最新数据

### 因子失效
运行 `2. 因子研发 -> 4. 因子监控` 检查因子状态

### 系统异常
运行 `6. 系统管理 -> 6. 系统健康检查` 诊断问题

### 推送失败
检查 `config/feishu_config.json` 配置是否正确
