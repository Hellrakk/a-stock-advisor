## 1. 估值因子

### PE_TTM（市盈率）

**定义：** 股价 / TTM每股收益

**行业标准：**
- 整体市场：15-25倍
- 银行业：5-10倍
- 科技股：30-50倍
- 消费行业：20-40倍

**低估判断：** < 行业平均 × 0.7

**高估判断：** > 行业平均 × 1.5

**使用场景：**
- 低估价值股筛选
- 风险评估（PE过高泡沫大）

**计算公式：**
```
PE_TTM = 当前股价 / (最近4季度净利润 / 总股本)
```

### PB（市净率）

**定义：** 股价 / 每股净资产

**行业标准：**
- 整体市场：1.5-3.0倍
- 银行业：0.5-1.2倍
- 科技股：3.0-8.0倍
- 消费行业：2.0-5.0倍

**低估判断：** < 行业平均 × 0.7

**高估判断：** > 行业平均 × 1.5

**使用场景：**
- 低估价值股筛选
- 资产质量评估

**计算公式：**
```
PB = 当前股价 / 净资产 / 总股本
```

### PEG（市盈增长率）

**定义：** PE / 盈利增长率

**行业标准：**
- < 1.0：低估
- 1.0-1.5：合理
- > 1.5：高估

**使用场景：**
- 成长股价值评估
- PE和增长率平衡

**计算公式：**
```
PEG = PE_TTM / 营收增长率
```

### 股息率

**定义：** 年股息 / 当前股价

**行业标准：**
- > 4%：高股息，适合防御
- 2-4%：正常
- < 2%：低股息

**使用场景：**
- 防御型配置
- 现金流稳定公司

**计算公式：**
```
股息率 = 最近一年股息总额 / 当前股价 × 100%
```

## 2. 盈利能力因子

### ROE（净资产收益率）

**定义：** 净利润 / 净资产

**行业标准：**
- 整体市场：8-12%
- 银行业：10-15%
- 科技股：15-25%
- 消费行业：12-20%

**优秀判断：** > 行业平均 + 5%

**较差判断：** < 行业平均 - 5%

**使用场景：**
- 盈利能力筛选
- 行业对比分析

**计算公式：**
```
ROE = 净利润 / 净资产 × 100%
```

### ROA（总资产收益率）

**定义：** 净利润 / 总资产

**行业标准：**
- 整体市场：3-6%
- 银行业：1-2%
- 科技股：5-10%
- 消费行业：5-8%

**使用场景：**
- 资产效率评估
- 轻重资产判断

**计算公式：**
```
ROA = 净利润 / 总资产 × 100%
```

### 毛利率

**定义：** （营收 - 成本） / 营收

**行业标准：**
- 整体市场：15-25%
- 银行业：不适用
- 科技股：40-70%
- 消费行业：20-40%

**竞争优势判断：** > 行业平均 + 10%

**使用场景：**
- 竞争优势评估
- 成本优势判断

**计算公式：**
```
毛利率 = （营业收入 - 营业成本） / 营业收入 × 100%
```

## 3. 成长性因子

### 营收增长率

**定义：** （本期营收 - 同期营收） / 同期营收

**行业标准：**
- > 20%：高成长
- 10-20%：成长
- 0-10%：平稳
- < 0%：衰退

**使用场景：**
- 成长股筛选
- 行业景气度判断

**计算公式：**
```
营收增长率 = (本期营收 - 同期营收) / 同期营收 × 100%
```

### 净利润增长率

**定义：** （本期净利 - 同期净利） / 同期净利

**行业标准：**
- > 30%：高速成长
- 10-30%：成长
- 0-10%：平稳
- < 0%：下滑

**使用场景：**
- 盈利成长性判断
- 规模化能力评估

**计算公式：**
```
净利润增长率 = (本期净利润 - 同期净利润) / 同期净利润 × 100%
```

### ROE增长率

**定义：** ROE同比增长

**行业标准：**
- > 5%：改善
- 0-5%：平稳
- < 0%：下滑

**使用场景：**
- 盈利改善判断
- 持续性评估

## 4. 技术面因子

### 动量20日

**定义：** (现价 - 20日前价格) / 20日前价格

**使用标准：**
- > 5%：买入信号
- < -5%：卖出信号
- -5% ~ 5%：观察

**使用场景：**
- 趋势跟踪
- 短期方向判断

### RSI

**定义：** 相对强弱指标

**使用标准：**
- < 30：超卖，买入机会
- 30-70：正常
- > 70：超买，卖出机会

**计算周期：** 14日

### MACD

**定义：** 指数平滑异同

**使用标准：**
- 金叉（DIF上穿DEA）：买入信号
- 死叉（DIF下穿DEA）：卖出信号

### 成交量比

**定义：** 当日成交量 / 5日均量

**使用标准：**
- > 1.5：放量，关注
- 0.5-1.5：正常
- < 0.5：缩量，谨慎

## 5. 情绪因子

### 融资融券余额

**定义：** 融资余额 / 融券余额

**使用场景：**
- 市场情绪判断
- 杠杆资金流向

### 北向资金流向

**定义：** 当日北向资金净流入/流出

**使用标准：**
- > 50亿：强烈买入
- 20-50亿：买入
- 0-20亿：中性
- < -20亿：卖出
- < -50亿：强烈卖出

## 6. 另类因子

### 分析师评级

**定义：** 分析师买入/维持/卖出评级数量

**使用标准：**
- 买入占比 > 60%：强烈看好
- 买入占比 40-60%：看好
- 买入占比 20-40%：中性
- 买入占比 < 20%：谨慎

### 机构持仓比例

**定义：** 机构持股占总股本比例

**使用标准：**
- > 30%：机构认可
- 20-30%：正常
- < 20%：散户主导
