## 一、买卖方向与价格区间

### 1.1 买入指令

**触发条件：**
- 新选股结果出现
- 现金比例 > 20%
- 持仓数量 < 7只

**执行标准：**
```
买入方向：买入（开仓）
买入价格区间：开盘价 ± 2%
买入数量：向下取整到100股的整数倍
买入时间窗口：
  - 首选：开盘后30分钟内（9:30-10:00）
  - 次选：收盘前30分钟（14:30-15:00）
```

**示例：**
```
股票：华能国际（600011）
目标权重：12%
计划金额：120,000元
开盘价：7.21元
可接受价格区间：7.07元 ~ 7.35元
计划买入数量：16,500股（向下取整）
实际下单：16,500股 @ 7.21元 = 118,965元
```

### 1.2 卖出指令

**触发条件：**
- 止盈触发（收益 > 20%）
- 止损触发（亏损 < -10%）
- 时间触发（持仓 > 60天）
- 因子触发（α下降 > 20%）

**执行标准：**
```
卖出方向：卖出（清仓/减仓）
卖出价格区间：开盘价 ± 1%（止损可放宽到±3%）
卖出数量：全部清仓或减仓1/3、减仓2/3
卖出时间窗口：
  - 止损：立即执行（不限时间）
  - 止盈：收盘前30分钟（14:30-15:00）
  - 换仓：次日开盘后30分钟内（9:30-10:00）
```

## 二、换仓规则

### 2.1 换仓触发条件

**止盈换仓：**
```
触发：收益 > 20%
动作：分3批止盈
  - 第一批（1/3）：触发当日收盘前30分钟
  - 第二批（1/3）：次日开盘后30分钟内
  - 第三批（1/3）：次日收盘前30分钟
替代：从当前选股池中选择最高α得分股票
```

**止损换仓：**
```
触发：亏损 < -10%
动作：立即全部清仓（强制执行）
时间：触发后5分钟内完成
替代：立即从选股池中选择替代股票
```

**时间换仓：**
```
触发：持仓 > 60天
动作：评估换仓（非强制）
评估标准：
  - α得分变化（相对建仓时）
  - 基本面变化
  - 行业轮动
决策：继续持有/减仓/清仓
```

**因子换仓：**
```
触发：α得分下降 > 20%（相对建仓时得分）
动作：建议换仓（非强制）
替代：从选股池中选择高α股票
执行时间：次日开盘后30分钟内
```

### 2.2 换仓执行流程

```
1. 检测触发条件（系统自动）
2. 记录触发时间和原因
3. 从当前选股池中选择替代股票
4. 生成换仓指令（卖出A、买入B）
5. 执行卖出（按卖出规则）
6. 等待资金到账（T+1）
7. 执行买入（按买入规则）
8. 更新持仓记录
```

## 三、止盈止损规则

### 3.1 止盈规则（详细）

**触发阈值：收益 > 20%**

**分批止盈：**
```
第一批（1/3仓位）：
  - 触发时间：当日14:30-15:00
  - 执行方式：市价卖出
  - 剩余仓位：2/3
  
第二批（1/3仓位）：
  - 触发时间：次日9:30-10:00
  - 执行方式：市价卖出
  - 剩余仓位：1/3
  
第三批（1/3仓位）：
  - 触发时间：次日14:30-15:00
  - 执行方式：市价卖出
  - 剩余仓位：0（全部清仓）
```

**止盈后资金分配：**
- 优先补充现金到20%
- 剩余资金按当前选股权重分配

### 3.2 止损规则（详细）

**触发阈值：亏损 < -10%**

**立即清仓：**
```
执行方式：立即市价卖出（不限时间）
执行时限：触发后5分钟内完成
滑点容忍：成交价不超过止损价的±3%
记录：止损原因、时间、成交价
```

**止损后处理：**
- 记录止损股票（暂缓买入30天）
- 更新选股池（排除该股票）
- 重新计算组合权重

## 四、持仓监控系统

### 4.1 风险指标监控

**波动率监控：**
```
指标：组合20日波动率
阈值：< 25%（年化）
预警：> 20%时警告
动作：> 25%时降低仓位
更新频率：每日收盘后
```

**VaR监控：**
```
指标：95%置信度VaR（10日）
阈值：< 15%
预警：> 10%时警告
动作：> 15%时暂停买入
更新频率：每日收盘后
```

### 4.2 行业偏离度监控

**指标：**
```
偏离度 = 组合行业权重 - 基准行业权重
阈值：< 20%（单个行业）
预警：> 15%时警告
动作：> 20%时调整持仓
更新频率：每日收盘后
```

**示例：**
```
行业：金融
基准权重：25%
组合权重：40%
偏离度：+15%（警告）
动作：考虑减持金融股
```

### 4.3 个股集中度监控

**指标：**
```
集中度 = 单股权重 / 总仓位
阈值：< 15%
预警：> 12%时警告
动作：> 15%时减持
更新频率：实时监控
```

## 五、市场状态监控

### 5.1 极端情况监控

**千股跌停监控：**
```
指标：跌停股票数量
触发阈值：> 1000只（占市场>20%）
动作：
  - 立即暂停所有买入
  - 暂停卖出（避免恐慌）
  - 记录市场异常
  - 等待市场恢复
```

**流动性枯竭监控：**
```
指标：成交量 < 5日均量的30%
触发阈值：组合内>50%股票流动性不足
动作：
  - 暂停所有交易
  - 等待流动性恢复
  - 记录异常情况
```

### 5.2 市场情绪监控

**恐慌指数：**
```
指标：VIX中国（如有）或 涨跌停比例
触发阈值：涨跌停比例 > 5:1（跌停远多于涨停）
动作：
  - 降低仓位至60%
  - 增加现金至40%
  - 暂停新增买入
```

## 七、执行流程

### 7.1 每日执行流程

```
08:00 - 执行盘前推送
  ├─ 检查交易日
  ├─ 加载最新数据
  ├─ 执行选股
  ├─ 生成报告（含买卖方向、价格区间）
  └─ 发送推送

09:30-10:00 - 执行买入
  ├─ 检查买入信号
  ├─ 确认价格区间（开盘价±2%）
  ├─ 执行买入（记录成交价、数量）
  └─ 更新持仓记录

14:30-15:00 - 执行卖出
  ├─ 检查卖出信号（止盈/止损）
  ├─ 确认价格区间（开盘价±1%）
  ├─ 执行卖出（记录成交价、数量）
  └─ 更新持仓记录

18:30 - 执行日报推送
  ├─ 更新持仓状态
  ├─ 计算风险指标
  ├─ 检查市场状态
  ├─ 生成报告（含风险监控）
  └─ 发送推送
```
